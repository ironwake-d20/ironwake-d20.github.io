## Critical Threshold

```
CT = 15 + Defender Armour Tier − Attacker Precision Bonus
```

**Assassin exception (undetected):**

```
CT = 10 − Attacker Precision Bonus
```

## Overdraw degradation

| Overdraw events | Approximate kai loss | Practical effect                          |
| --------------- | -------------------- | ----------------------------------------- |
| 1               | ~1–2% of tier        | Barely perceptible mana reduction         |
| 3–5             | ~5–8% of tier        | Noticeable pool shrinkage                 |
| 10+             | ~15–25% of tier      | Possible tier boundary risk; feature loss |
| Catastrophic    | Tier drop            | Effectively losing a class level ceiling  |
