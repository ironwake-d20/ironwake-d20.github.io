# Weapons

Every weapon has a **base damage die** and may have a **Precision bonus** — the ability to bypass armour and find gaps. Precision is weapon-gated: a stiletto can have high Precision; a greatsword cannot.

## Weapon properties

| Property   | Effect                                                    |
| ---------- | --------------------------------------------------------- |
| Finesse    | Use DEX instead of STR for attack and damage rolls        |
| Light      | Can be used for two-weapon fighting                       |
| Heavy      | Small creatures have disadvantage                         |
| Reach      | +5 ft. melee range                                        |
| Two-handed | Requires both hands                                       |
| Loading    | Can fire only once per action regardless of extra attacks |
| Thrown     | Can be thrown; use the weapon's damage die                |

## Precision weapons

These weapons are designed to bypass armour. They have a Precision bonus that reduces the Critical Threshold (see [Combat](../core-rules/combat.md)):

| Weapon       | Base die | Precision | Notes                                      |
| ------------ | -------- | --------- | ------------------------------------------ |
| Stiletto     | d4       | +2        | Concealable; designed for gaps in plate    |
| Estoc        | d6       | +2        | Two-handed thrusting sword                 |
| Bodkin arrow | d6       | +1        | Requires a bow; armour-piercing ammunition |
