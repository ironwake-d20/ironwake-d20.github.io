# Attributes

The six attributes define a character's raw physical and mental capabilities. Each attribute has a **score** (typically 3–18 for starting characters) and a **modifier** derived from that score.

## Attribute modifier

```
Modifier = (Score − 10) ÷ 2, rounded down
```

| Score | Modifier |
| ----- | -------- |
| 3     | −4       |
| 4–5   | −3       |
| 6–7   | −2       |
| 8–9   | −1       |
| 10–11 | +0       |
| 12–13 | +1       |
| 14–15 | +2       |
| 16–17 | +3       |
| 18–19 | +4       |
| 20–21 | +5       |
| 22-23 | +6       |
| 24-25 | +7       |

## The six attributes

### Strength (STR)

Physical power. Governs melee attack rolls, damage with non-finesse weapons, carrying capacity, and feats of athletics.

### Dexterity (DEX)

Agility and reflexes. Governs ranged attack rolls, finesse weapon attacks, initiative, stealth, and acrobatics.

### Constitution (CON)

Endurance and resilience. Determines **Vitality** maximum (with class) and **Wounds** (CON ÷ 4, minimum 1). Governs resistance to poison, disease, and fatigue.

### Intelligence (INT)

Reasoning and knowledge. Determines **mana pool** size and maximum **spell level**. Governs arcana, investigation, and lore skills.

### Wisdom (WIS)

Perception and intuition. Governs awareness, insight, survival, and resistance to mental influence.

### Charisma (CHA)

Force of personality. Governs persuasion, deception, intimidation, and social standing.
