# Skills

Skills represent training, practice, and expertise in specific areas. They are used in conjunction with ability checks when a character attempts something that falls within their training.

## Skill points

Skill points are determined by your kai tier band at character creation:

| Band             | Skill points  |
| ---------------- | ------------- |
| Low (x000–x299)  | Base for tier |
| Mid (x300–x699)  | Base + 2      |
| High (x700–x999) | Base + 2      |

The high-band bonus also grants +1 to one class-relevant skill check category.

## Proficiency

When you are proficient in a skill, you add your **proficiency bonus** to checks using that skill. Proficiency bonus scales with character level.
